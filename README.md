# 引き継ぎドキュメント
## AIコンサルタント 松竹梅プラン設計ツール

---

## プロジェクト概要

SHIFT AI「AIコンサルタントVer3」受講生向けに、自分のスキルと現状を入力するだけで松竹梅の3プランと価格を自動生成できるWebツール。HTML単体ファイルで動作し、外部API不使用・完全無料。

**成果物**：`AIコンサル_松竹梅プラン設計ツール.html`（単体HTMLファイル）
**配布方法**：GitHub Pages / Netlify 等でURL共有予定

---

## ベースとなる講義内容（Day9骨子より）

- **価格公式**：Alan Weissの価値ベース価格設定
  - 総リターン（見える価値＋見えない価値）の10〜30%が報酬目安
  - 竹の月額 = 総リターン × 20% ÷ 12ヶ月
- **4段階ロードマップ**
  - STEP1：無料で3件（信頼・実績作り）
  - STEP2：低価格で受注（月1万〜）
  - STEP3：成果報酬（リターンの10〜30%）
  - STEP4：高額提案・顧問契約（月50万〜）
- **松竹梅の心理効果**：アンカリング効果・ゴルディロックス効果
- **ROI計算例**：月50時間削減 × 12ヶ月 × 2,000円 = 年120万（見える価値）＋離職防止100万（見えない価値）= 総リターン220万 → 竹の月額 ≒ 3.6万円

## アプリの画面構成・フロー

### 全体フロー

```
STEP1（ステージ選択）
  ↓
  ├─ ステージ1 → STEP2（スキル選択）→ 梅プランのみ表示
  └─ ステージ2〜4 → STEP2（スキル選択）→ STEP3（ROI入力）→ 松竹梅プラン表示
```

---

### STEP1｜ステージ選択

| 選択肢 | 内容 |
|--------|------|
| ステップ1 | まだ案件がない（これから始める） |
| ステップ2 | 低単価で動いている（月1〜3万円程度） |
| ステップ3 | 受注できているが単価を上げたい |
| ステップ4 | 高額案件・顧問契約を狙っていきたい |

---

### STEP2｜スキル選択（全ステージ共通）

チェックリスト（複数選択可）＋自由記述（任意）

- ChatGPT / 生成AI活用支援
- 業務効率化・自動化
- Googleワークスペースの活用
- Excel / スプレッドシート活用
- 営業・商談サポート
- マネジメント・組織支援
- SNS・情報発信支援
- 採用・人材育成支援
- その他（自由記述）

---

### STEP3｜ROI計算インプット（ステージ2〜4のみ）

**① 地域選択 → 実質時給を自動セット（手動上書き可）**

| 地域 | 実質時給 |
|------|---------|
| 三大都市圏（東京・大阪・名古屋） | 2,500円 |
| 地方都市（政令指定都市など） | 2,000円 |
| 地方・郡部 | 1,600円 |

**② 事業規模 → 対象人数を自動セット（手動上書き可）**

| 規模 | 想定人数 |
|------|---------|
| 小規模（〜10人） | 5人 |
| 中規模（10〜50人） | 20人 |
| 中大規模（50〜100人） | 40人 |
| 大規模（100人以上） | 80人 |

**③ 月間削減時間／人 → 3択＋手動入力**

| 選択肢 | 時間 | イメージ |
|--------|------|---------|
| 軽度 | 月5h | 情報検索・質問対応の効率化 |
| 中程度 | 月15h | 定型業務の半自動化・資料作成 |
| 大幅 | 月30h | 主要業務フローの自動化 |

**④ 見えない価値**（任意・万円単位）：離職防止・評判向上など

---

### ROI計算ロジック

```
見える価値（年間） = 削減時間/人/月 × 対象人数 × 12ヶ月 × 実質時給
見えない価値       = 入力値（万円 × 10000）
総リターン         = 見える価値 + 見えない価値

竹の月額   = 総リターン × 20% ÷ 12（千円単位切り上げ）
松の月額   = 竹 × 1.7倍（千円単位切り上げ）
梅の月額   = 竹 × 0.4倍（千円単位切り下げ・最低5,000円）
```

---

### STEP4｜プラン出力

**ステージ別の表示ルール**

| ステージ | 表示プラン | 推奨バッジ |
|---------|-----------|----------|
| 1 | 梅のみ（3件実績後に提案する形で表示） | なし |
| 2 | 梅・竹 | 竹「おすすめ」 |
| 3 | 松・竹・梅 | 竹「本命」 |
| 4 | 松・竹・梅 | 松「狙い目」 |

**ステージ1の特別表示**
- 「有料化までの流れ」として①無料3件→②実績獲得→③このプランで有料化、という3ステップを表示
- 梅プランは「3件の実績後に提案するプラン」として金色ボーダーで強調表示
- スキルで選んだ内容は「対応領域：〇〇・〇〇」として1行にまとめて表示（全部並べてセッション回数が膨れて見えるのを防ぐ）

**全ステージ共通の締めメッセージ**
> 「提供できるスキルが増えたとき、クライアントの規模が大きくなったとき、次のステージに進んだとき——そのたびにこのツールでプランを見直してみてください。あなたの価値は、これからどんどん上がっていきます💪」

**その他機能**
- ROI計算根拠のアコーディオン表示（「この価格の計算根拠を見る」）
- 画像ダウンロードボタン（html2canvas使用）
- 最初からやり直すボタン

## これまでに対応した修正履歴

1. チェックボックスがクリックしても反応しない → `onchange`イベントベースに修正
2. ステージ1でもスキル選択を通るように変更（スキップしていたのを修正）
3. 「断られることはありません」という文言を削除 →「小さく始めることが大切です」に変更
4. 梅プランでスキルを全部羅列→月何回もやるように見える問題を修正 → スキル名を「対応領域」として1行にまとめる形に変更
5. ステージ1の表示を「無料3件やったらまたツールに戻ってきて」から「3件の実績後にこのプランで有料化しましょう」という流れに変更
